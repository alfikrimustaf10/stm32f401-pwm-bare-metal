#stm32f401 PWM simple project bare metal

#features 
-pwm led control
-interrupt button
-breadboard (optional)

#structure
-include
-src
-core
-interrupt
-linker
-startup

#toolchain
-arm-none-eabi-gcc (if you are using arch linux, you can install via sudo pacman -S arm-none-eabi-gcc arm-none-eabi-binutils)
-vim (optional)

#note 
require intermediate to advanced C progamming knowledge, microcontroller architecture especially stm32, know how to read documentation. if you just started your jorney on microcontoller, i,d recomend to use stm32 cube ide.
